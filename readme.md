##  webpack+gulp+react+react-router+issues+webhoot 构建的个人博客系统

> 作者：李成龙 <lichenglong@kongfz.com>

### 环境依赖

node v8.4.0+


### 部署步骤

```
1. npm install        //安装node运行环境
2. npm run dev        //开发环境构建
3. npm run start      //开发环境构建
4. npm run build      //生产环境构建
```
